data-diff
---------

**Data-diff** is a command-line tool and Python library for comparing tables in and across databases.

This repository is a maintained fork by the Gannett / USA TODAY LOCALiQ Data Engineering team.
The project was originally developed by Datafold.

For installation, examples, and CLI usage, see the `README <https://github.com/GannettDigital/data-diff#readme>`_.
